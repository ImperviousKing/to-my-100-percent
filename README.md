An interactive web application created to celebrate a first anniversary. This progressive web app features a journey through memories with challenges, a photo gallery, and a heartfelt message.

## Features

- **Loading Screen**: Progressive loading with percentage display
- **Interactive Stages**: Multiple stages with challenges and riddles
- **Photo Gallery**: Slideshow of memories with captions
- **Video Support**: Special video slide with custom audio
- **Memory Collage**: Cebu trip photo collage
- **Background Music**: Customizable music controls
- **Progress Tracking**: Visual progress ring showing completion percentage
- **Responsive Design**: Works on desktop and mobile devices

## Usage

1. Open `index.html` in a web browser
2. Wait for the loading screen to complete (100%)
3. Follow the interactive journey through the stages
4. Complete challenges to unlock memories
5. View the photo gallery and collage
6. Read the final anniversary message

## Browser Support

Works best on modern browsers that support:
- CSS Grid
- Flexbox
- HTML5 Video/Audio
- Service Workers (for PWA features)

## Notes

- Background music may require user interaction to play (browser autoplay policies)
- Use the music toggle button to control background music
- The app is designed as a Progressive Web App (PWA) and can be installed on devices

## Customization

To customize the content:
- Edit the stage questions and answers in `index.html`
- Update image sources to match your file names
- Modify the final message in the "FINAL SCREEN" section
- Adjust colors in the CSS (currently using purple/lavender theme)
